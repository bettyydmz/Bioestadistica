### 📊 Bioestadística aplicada en R

Repositorio que documenta análisis y ejercicios desarrollados durante mi formación en bioestadística, enfocados en la exploración, modelado y visualización de datos biológicos usando R.

---

## 🚀 Descripción del proyecto

Este repositorio reúne distintos análisis reproducibles realizados en R y Quarto, abordando conceptos fundamentales de bioestadística mediante datos reales y simulados.

Los notebooks incluyen desde estadística descriptiva hasta el uso de distribuciones de probabilidad y análisis exploratorio de datos (EDA), con énfasis en la interpretación biológica de los resultados.

---

## 🧠 Habilidades demostradas

- Análisis exploratorio de datos (EDA)
- Manejo de distribuciones (Poisson, continuas, etc.)
- Visualización de datos en R
- Interpretación estadística aplicada a biología
- Uso de R Markdown y Quarto para reproducibilidad

---

## 🗂️ Estructura del repositorio

| Archivo | Descripción |
|--------|------------|
| `ClaseR.Rmd` | Introducción a R y manipulación de datos |
| `Dist_Vas.Rmd` | Distribuciones de variables aleatorias |
| `Dist_Vas_Continuas.qmd` | Modelado de variables continuas |
| `Dist_Vas_modificada.Rmd` | Ajustes y variaciones de modelos |
| `EDA_penguins.qmd` | Análisis exploratorio del dataset *penguins* |
| `Betty_Seeds_pumpkins.qmd` | Caso de estudio aplicado |
| `dist_poisson.Rproj` | Proyecto enfocado en distribución Poisson |
